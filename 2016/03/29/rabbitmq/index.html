centOs 6.4安装rabbitMq

centOs6.4坑比较多，

  1. yum安装nc会报错Procotol not，解决办法使用旧版的nc重新安装。

    rpm -ivh http://vault.centos.org/6.6/os/x86_64/Packages/nc-1.84-22.el6.x86_64.rpm

  2. 没有直接yum安装的rabbitMq

安装编译环境，

yum install make gcc gcc-c++ kernel-devel m4 -y

安装git

yum install git

安装erlang
下载包并编译

下载rabbitmq包

  1. git clone https://github.com/rabbitmq/rabbitmq-server.git
  2. cd rabbitmq-server
  3. make
  4. cd script
  5. ./rabbitmq-server
faq

有时候直接运行./rabbitmq-server报错

ERROR: epmd error for host "****": timeout (timed out establishing tcp connection)

解决办法：
vi /etc/hosts
增加主机名与127.0.0.1的绑定
